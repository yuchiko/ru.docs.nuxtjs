# Добавление PostCSS плагинов

В вашем файле конфигурации `nuxt.config.js` укажите:

```js
module.exports = {
  build: {
    postcss: [
      'postcss-nested': {},
      'postcss-responsive-type': {},
      'postcss-hexrgba': {},
    ]
  }
}
```
